-- SUMMARY --

The Signage Reload module does a full page load for signage sign entities every
10 minutes.

Before the reload, the module first checks for connectivity by pinging itself
using the hostReachable javascript library.


-- REQUIREMENTS --

* Libraries API - http://drupal.org/project/libraries
* hostReachable js gist - https://gist.github.com/brandonneil/6921434


-- INSTALLATION --

* Install as usual, see
  http://drupal.org/documentation/install/modules-themes/modules-7
